Welcome to bosk documentation!
================================

Bosk is the Python framework for creating and maintaining deep forest models. It also allows you to create general-purpose computational graphs,
consisting of predefined or mainly created blocks, and compare different models efficiency via cross-validation.

To get basics, read the :doc:`user guide <./user_guide>`.

Installation
---------------
To install bosk just run the following command in the root of the bosk's source code:

.. code-block:: console
   
   pip install -r requirements.txt
